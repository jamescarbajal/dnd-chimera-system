# dnd-chimera-system




Chimera Catalogue 
Supplementary system of unlockable and customizable magical traits/spells/abilities based on bloodline.





BOI'S CHIMERIC CATALOGUE — GM CODE REFERENCE
=====================================================

Keep this file private. Each code unlocks exactly one Trait.
Codes are case-insensitive. The player enters them in the
"Enter unlock code" field at the top of the Catalogue tab.

STATUS KEY:  [UNLOCKED] = available by default
             [LOCKED]   = requires code entry to unlock


=====================================================
MASTER CODE
=====================================================
CHM-D3F8-49D8   — Unlocks ALL locked built-in Traits at once.
                  Enter again to lock all built-in Traits.
                  Does NOT affect custom Traits the player has created.

=====================================================
EPISTASIS COMBINATIONS
=====================================================

Iron Covenant		        Null Grasp + Radiant Vow		            Cannot be reduced below 1HP by a single strike.

Planar Convergence	    Blink + any Conjuration trait		        Pull any Conjuration Trait into a phantom 4th Pool slot.

The Unveiled Thread	    Fated Strike + Foretell Doom		        Ask the GM one yes/no question; they must answer truthfully.

Sovereign Gaze		      Hex Bolt + Compel Truth			            One creature is charmed for up to one hour, no save

Cascade			            Combust + Arc Flash			                Your next damage roll applies to all creatures within 10ft of the target

Fractured Mirror	      Terrifying Visage + Veil of Mist	      Three illusory duplicates appear; each vanishes on contact, lasting up to one hour

Sanguine Debt		        Grave Chill + Death Knell		            Regain HP equal to your next damage roll total

Crucible State		Pounce + Iron Skin			Instantly swap any Pool Trait for a Transmutation Trait from your Catalogue
